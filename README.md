# PHP User Management System

A simple User Management System developed using PHP, HTML, CSS, and XAMPP.

## Features
- User Registration
- User Login Authentication
- Manage Users Dashboard
- Edit User Information
- Delete User Records
- CRUD Operations (Create, Read, Update, Delete)

## Technologies Used
- PHP
- HTML5
- CSS3
- XAMPP

## Project Objectives
- Implement user authentication.
- Manage user data efficiently.
- Perform CRUD operations.
- Gain hands-on experience in backend web development.

## How to Run
1. Install XAMPP.
2. Start Apache.
3. Place the project folder inside htdocs.
4. Open http://localhost/project-folder-name in your browser.
